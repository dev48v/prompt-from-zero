🪞 Self-RAG — the Model Decides When to Retrieve, Then Grades Itself

Not every question needs a search. Self-RAG knows the difference.

Plain RAG retrieves for everything. Self-RAG makes the model emit "reflection tokens": Retrieve? first, then grade each doc Relevant?, and grade its own answer Supported? / Useful? — retrying if it falls short. Pick a question:

Four small LLM judgements wrapped around your normal retrieve-and-generate.

    Decide: retrieve at all?
    const needs = await llm(`Does answering this need external/private facts?
Answer RETRIEVE or NO_RETRIEVE. Question: ${q}`);
if (needs === "NO_RETRIEVE") return llm(q);   // skip the whole search
