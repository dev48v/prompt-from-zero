🔧 Corrective RAG (CRAG) — Grade the Retrieval, Then Fix It

If the retrieved docs are wrong, don't answer from them — correct course.

CRAG adds a self-grading step after retrieval: a retrieval evaluator scores every candidate doc. Correct → use the docs. Incorrect → discard them + run a web search for fresh knowledge. Ambiguous → combine both. Then a knowledge-refinement step strips the irrelevant lines before generating. Pick a question:

🔨 Add CRAG to your pipeline

    
Wrap your normal retrieve-and-generate with a grader and three corrective branches.

    
    
      
  1. 1
    Retrieve candidate docs as usual
    const docs = await vectorStore.search(q, { k: 4 });
// these are CANDIDATES — we don't trust them yet.
    2. 
      
  2. 2
    Grade each doc with an evaluator
    // a small LLM or a fine-tuned classifier returns a 0..1 score
async function gradeDoc(q, d) {
  const out = await llm(`Score 0.0-1.0 how RELEVANT this passage is
to the question. Reply with just the number.
Q: ${q}
PASSAGE: ${d}`);
  return parseFloat(out);                 // e.g. 0.91
}
const scored = await Promise.all(docs.map(async d => ({ d, s: await gradeDoc(q, d) })));
    3. 
      
  3. 3
    Pick a verdict from the scores
    const top = Math.max(...scored.map(x => x.s));
const verdict =
  top >= 0.75 ? "CORRECT"   :   // at least one strong doc
  top <  0.40 ? "INCORRECT" :   // nothing is relevant
                "AMBIGUOUS";    // weak / partial — unsure
    4. 
      
  4. 4
    Branch: correct course per verdict
    let knowledge = [];
if (verdict === "CORRECT")   knowledge = scored.filter(x => x.s >= 0.5).map(x => x.d);
if (verdict === "INCORRECT") knowledge = await webSearch(q);         // discard + search
if (verdict === "AMBIGUOUS") knowledge = [                            // combine both
  ...scored.filter(x => x.s >= 0.5).map(x => x.d),
  ...await webSearch(q),
];
    5. 
      
  5. 5
    Refine: decompose → filter → recompose
    // split each doc into strips, keep only the relevant strips
function refine(q, doc) {
  const strips = doc.split(/(?<=[.!?])\s+/);                 // sentence strips
  return strips.filter(s => gradeStrip(q, s) > 0.5).join(" ");
}
knowledge = knowledge.map(d => refine(q, d)).filter(Boolean);
    6. 
      
  6. 6
    Generate from the corrected knowledge
    const answer = await llm(`Answer using ONLY this knowledge:
${knowledge.join("\n---\n")}

Question: ${q}`);
return answer;   // grounded in corrected, refined context
