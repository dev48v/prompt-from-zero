🔨 Build it yourself

    
Tool-augmented ReAct = define tools → write system prompt → run the loop → parse tool calls → feed results back. Repeat.

    
    
      
  1. 
        
    1
    Define your tool list in the system prompt
    
        
    The model must know what tools exist, what each one does, and the exact format to call them. Put this in the system prompt.
    
        
    You are a ReAct agent. For EVERY step write:
Thought: <your reasoning>
Action: <tool_name>[<input>]
  OR (when ready)
Final Answer: <answer>

Available tools:
- calculator[expr]  — evaluates a math expression, e.g. calculator[23*47]
- search[query]     — looks up a factual question, e.g. search[population of France]

Never skip the Thought. Never skip the Action format.
    
      
    2. 

      
  2. 
        
    2
    Parse the model's output for tool calls
    
        
    After every model reply, check whether it wrote an Action: line. If yes — run the tool. If it wrote Final Answer: — stop.
    
        
    function parseAction(text) {
  // matches: Action: calculator[23*47]
  const m = text.match(/Action:\s*(\w+)\[(.+?)\]/);
  if (m) return { tool: m[1], input: m[2] };
  if (/Final Answer:/i.test(text)) return { tool: "done" };
  return null;
}

    Run the tool and feed the Observation back
    
        
    Append the tool result as an Observation: line in the conversation, then re-call the model. The model now reasons on real data.
    
        
    async function runTool(tool, input) {
  if (tool === "calculator") {
    // safe math eval
    return String(Function(`return (${input})`)());
  }
  if (tool === "search") {
    return await fetchSearchResult(input); // your real or mock API
  }
}

// In the loop:
const obs = await runTool(action.tool, action.input);
messages.push({ role: "user", content: `Observation: ${obs}` });
// then call the model again

    4
    The full loop — run until Final Answer
    
        
    const messages = [{ role: "user", content: question }];
let answer = null;

for (let step = 0; step < 8; step++) {
  const reply = await llm(systemPrompt, messages);
  messages.push({ role: "assistant", content: reply });

  const action = parseAction(reply);
  if (!action) break;                        // safety: no recognised pattern
  if (action.tool === "done") {
    answer = reply.match(/Final Answer:\s*(.+)/)?.[1];
    break;
  }

  const obs = await runTool(action.tool, action.input);
  messages.push({ role: "user", content: `Observation: ${obs}` });
}

console.log("Answer:", answer);
