🗳️ Self-Consistency

Same hard math problem → sample 5 Chain-of-Thought attempts → take majority vote. Watch outliers get filtered out.

A train leaves at 60 km/h. After 30 minutes, a second train leaves the same station at 80 km/h on the same track. How many minutes after the second train leaves do they meet?

Correct answer: 90 minutes. (First train has 30 km lead at 60 km/h. Second closes 20 km/h gap. 30 km ÷ 20 km/h = 1.5 h = 90 min.)

↑ Single CoT gets it wrong ~40% of the time. 5-sample majority gets it right ~95%. Cost: 5× the tokens.

    Write self-consistency.mjs
    
        
    

    import { generateText } from "ai";
import { google } from "@ai-sdk/google";

const model = google("gemini-2.5-flash");
const problem = "A train leaves at 60 km/h. 30 min later, a second train leaves the same station at 80 km/h. How long after the second train leaves do they meet? Let's think step by step.";

// Sample N times with high temperature for diversity.
const N = 5;
const samples = await Promise.all(
  Array.from({ length: N }, () =>
    generateText({ model, prompt: problem, temperature: 0.7 })
  )
);

// Extract numeric answer from each reasoning chain — last number in text.
function extract(text) {
  const nums = text.match(/-?\d+(?:\.\d+)?/g);
  return nums ? nums[nums.length - 1] : null;
}

const answers = samples.map(s => extract(s.text));
console.log("Samples:", answers);

// Tally + pick majority.
const tally = {};
for (const a of answers) tally[a] = (tally[a] || 0) + 1;
const [winner, votes] = Object.entries(tally)
  .sort((a, b) => b[1] - a[1])[0];

console.log(`Final (majority): ${winner} — ${votes}/${N} votes`);

    Tweaks
    
        
      
          
    • Bump N to 10 or 20 — better accuracy, more cost.
      • 
          
    • Lower temperature (0.4-0.6) to keep diversity but reduce wild outliers.
      • 
          
    • Add a confidence score: votes / N ≥ 0.6 = trust, else flag for human review.
      • 
          
    • For NON-numeric answers: normalize text (lowercase + trim + canonical form) before tallying.
